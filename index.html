Cater'd

Wep App

Cater'd is a fully functioning web app that another student and I built for our final project at The Iron Yard. It's a platform for connecting private chefs and cooks with their clients. Clients can create events for chefs to cater and chefs can create events for clients to attend.

Cater'd is built with the Django web framework and Python programming language on the back-end, using HTML5, CSS3, and Materialize for the front-end presentation. We incorporated Elasticsearch with Django-Haystack for the search index. We also successfully deployed the app to the web using Amazon Web Services.

Q&A

Web App

Q&A is a fully functioning web app that allows users to ask and answer questions on a variety of topics. This was a team project for The Iron Yard requiring us to build a Django web app patterned after the Stack Overflow website. Questions can be tagged with multiple subject tags and answers can be voted up or down.

Technologies used: Django, Python, HTML5, CSS3, Materialize
